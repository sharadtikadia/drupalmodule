INTRODUCTION
------------

Custom CSV module to import data by creating nodes programatically. 

STEPS
-----

1. Create modules/custom/modulename/modulename.info.yml file with details of modules. 
2. Create modules/custom/modulename/modulename.module file
3. Enable module from CMS admin extensions
4. Create modules/custom/modulename/module.routing.yml
5. Create Controller file in src folder. eg. modules/custom/modulename/src/Controller/IastorelocatorController.php Give name according to what you have mentioned in routing.yml file.
6. Download this module and modify it according to your requirement of import data.
