INTRODUCTION
------------

Custom CSV module to import data by creating nodes programatically. 

STEPS
-----

1. Create modulename.info.yml file with details of modules. 
2. Create modulename.module file
3. Enable module from CMS admin extensions
4. Create module.routing.yml
5. Create Controller file in src folder. Give name according to what mentioned in routing.yml file.
6. Download this module and modify it according to your requirement of import data.
