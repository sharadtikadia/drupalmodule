INTRODUCTION
------------

Custom CSV module to import data by creating nodes programatically. 

STEPS
-----

1. Create <b>modulename.info.yml</b> file with details of modules. 
2. Create <b>modulename.module</b> file
3. Enable module from CMS admin extensions
4. Create <b>module.routing.yml</b>
5. Create Controller file in <b>src</b> folder. Give name according to what mentioned in routing.yml file.
6. Download this module and modify it according to your requirement of import data.
